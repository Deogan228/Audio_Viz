# Audio-Viz

Визуализатор и анализатор аудио-файлов на Scala 3 в функциональном стиле.

## Возможности

- Чтение WAV-файлов (PCM, 16 бит, моно/стерео)
- Быстрое преобразование Фурье (FFT) для анализа частотного спектра
- Детекция битов по энергии звука
- Расчёт BPM (ударов в минуту)
- Визуализация спектра в терминале

## Архитектура

Проект построен на собственных монадах (без библиотек):

- **Reader[Config, A]** — проброс конфигурации (размер FFT, пороги детекции)
- **Writer[Vector[String], A]** — накопление логов анализа
- **State[DetectorState, A]** — состояние детектора битов
- **IO[A]** — взаимодействие с консолью и файловой системой

## Структура:
src/main/scala/
├── monads/          # Блок 0: собственные монады
├── domain/          # Предметные типы и алгоритмы
└── app/             # IO-сценарий

## Запуск

```bash
sbt run
```

Программа запросит путь к WAV-файлу и проведёт анализ.

## Требования

- JDK 17+
- sbt 1.9+
- Scala 3.3.4
